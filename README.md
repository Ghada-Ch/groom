This is a free, open-source project that helps people generate a clean, printable marriage profile while giving developers an easy way to contribute by adding translations.

## ✨ Features

- 🌍 Multiple languages
- 📄 Printable PDF
- 📱 Responsive design
- 🌙 Dark mode
- 💯 Free and open source

## 🚀 How to Contribute
Choose any unchecked language, translate index.html, and submit a Pull Request!
First-time contributors are always welcome. ❤️

1. Fork the repository.
2. Copy `index.html`.
3. Rename it using the language code (e.g. `index_fr.html`).
4. Translate all visible text.
5. Keep the HTML, CSS, and JavaScript unchanged.
6. Submit a Pull Request.
   
Read the [CONTRIBUTING.md](CONTRIBUTING.md) guide.

❤️ Every new translation helps make Suitor's Dossier accessible to more people around the world.
